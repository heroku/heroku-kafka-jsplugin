A plugin to manage Heroku Kafka.

## Install

``` sh-session
$ heroku plugins:install heroku-kafka
```

## Development

For normal development, the initial setup is:
``` sh-session
# ensure node 20.x or higher is installed
$ npm install
$ heroku plugins:link
```
